A list of resources I use frequently, mostly but not exclusively related to web development. 

## Interesting articles

- <Link href="https://kentcdodds.com/blog/aha-programming">AHA Programming</Link> - Adds AHA (Avoid Hasty Abstractions) to the list of programming acronyms like DRY (Don't Repeat Yourself) and WET (Write Everything Twice).
- <Link href="https://people.apache.org/~fhanik/kiss.html">Keep It Stupid Simple, or Keep It Simple, Stupid</Link> - This reads like a classic, and it sure looks like one.
- <Link href="https://moxie.org/2022/01/07/web3-first-impressions.html">@moxie's first impressions of web3</Link> - This is the creator of Signal and this essay really clarified the modern web3 situation for me. 

## (Academic) writing
- <Link href="https://hemingwayapp.com/">Hemingway Editor</Link> - Teaches you to write bold and clear language. 
- <Link href="https://click.endnote.com/invite/ec3642115">EndNote Click</Link> - Neat browser extension for Firefox and Chrome, that searches for full-text PDFs of research papers.

## Web design
- <Link href="https://freeillustrations.xyz/">Free Illustrations</Link> - A collection of collections of free SVG illustrations. 
- <Link href="https://cari.institute/">Cari Institute</Link> - Cari stands for "Consumer Aesthetics Research Institute", the website holds a collection of different aesthetics from the 1970s until now. 
- <Link href="https://toolkit.addy.codes/">Addy Codes Toolkit</Link> - Huge collection of tools and resources.

## Small tools
- <Link href="https://png-pixel.com/">PNG Pixel</Link> - Generates a tiny Base64 encoded pixel that you can use as a placeholder for things like lazy loading images.
- <Link href="https://previewed.app/">Previewed</Link> - Create free device mockups up to a certain resolution.
